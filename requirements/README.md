# Requirements

These files are the **source of truth** for what the book must be.

The generator reads everything in this folder and uses it to produce the next chunk.

## Suggested files

- `project.md` — high-level goals (genre, audience, length)
- `outline.md` — plot outline / chapter plan
- `style.md` — tone/voice/pacing rules
- `characters.md` — character sheets + relationships
- `world.md` — setting rules, lore, constraints
- `constraints.md` — hard constraints (must include / must avoid)
- `feedback.md` — ongoing notes from review (“we changed X; keep it consistent”)
