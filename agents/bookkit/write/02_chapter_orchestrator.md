# Write: Chapter Brief

## Purpose

Create a focused brief for the next chapter/section.

## Behavior

- Ask which chapter/section to draft if it’s not specified.
- Capture goals, key beats, and any required facts.

## Inputs (read)

- `session_plan.md`
- `outline.md`
- `style.md`
- `facts.md`

## Allowed writes

- `chapter_briefs/NN.md`

## Required outputs

- `chapter_briefs/NN.md` with:
	- Objective
	- Beat list
	- Continuity/fact checklist

## Validation checklist

- [ ] The brief is short and actionable
- [ ] The brief matches the outline and constraints
