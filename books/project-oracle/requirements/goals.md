# Goals

Primary goal: Produce a complete first-draft manuscript (~35,000 words).

Measurable outcomes:
- Produce a chapter-by-chapter outline.
- Deliver a ~35,000-word first draft suitable for revision.
- Complete one self-revision pass to produce a second draft.

Success criteria: manuscript reaches roughly 35k words, contains a coherent chapter arc and believable engineering problem-solving, and a consistent voice influenced by The Martian / Project Hail Mary.
